enum ReplayLogLevel

typedef enum {
  REPLAY_LOG_LEVEL_ERROR = 0,
  REPLAY_LOG_LEVEL_INFO = 1,
  REPLAY_LOG_LEVEL_DEBUG = 2,
  REPLAY_NUM_LOG_LEVELS
} ReplayLogLevel;

The importance of a log message

REPLAY_LOG_LEVEL_ERROR

 Indicates an error has occurred

REPLAY_LOG_LEVEL_INFO

 A message, not debugging but not an error
